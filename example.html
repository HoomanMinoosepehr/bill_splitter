## Requirement template for Bill Splitter App
  1. Navbar
    App Logo | Bill Splitter | Billing Index
  2. App Logo
    This page will show greeting message, information about your application, and you to use the application. You can also tax information. For example,

    Meals: 5% GST
    Alcoholic beverages: 10% PST + 5% GST

  3. Bill Splitter
    Restaurant Bill: (required)
    Tip(15% to 20%): (not required)
    Custom Tip: (not required)
    Tax: GST GST+PST(not required)
    Number of People: (required)
    Show the Information
    Billing Amount
    Tip
    Tax
    All-In Total:
    Tip Each Person:
    Tax Each Person:
    Total Each Person:
  4. Save the submitted billing information in db.
  5. Billing Index
    This page will show list submitted billing information with submission date and total billing amount. Clicking on a list will take to detail page showing all the information. In the detail page user can delete a bill.
  6. Optional/Stretch
    Add a button in the detail page of submitted billing info where user can mark a bill resolved or unresolved.
